## üö® QUICK TROUBLESHOOTING

### Problem: "Cannot connect to server"
**Solution**: Start server first (see START.bat or manual setup)

### Problem: "Module not found"
**Solution**: Run `pip install -r requirements.txt`

### Problem: "Account not found"
**Solution**: Use ACC001, ACC002, or ACC003

### Problem: "Insufficient balance"
**Solution**: Check balance first (Menu option 4)

‚Üí More details: **PROFESSIONAL_GUIDE.md** ‚Üí Troubleshooting
